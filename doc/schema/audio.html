Data structures

Struct: Sound

Key Field Type Description Requiredness Default value
1 wav binary optional
2 mp3 binary optional
3 sph binary optional
4 path string An absolute path to a file on disk where the sound file can be
found. It is assumed that this path will be accessable from any
machine that the system is run on (i.e., it should be a shared
disk, or possibly a mirrored directory).
optional

A sound wave. A separate optional field is defined for each suppported format. Typically, a Sound object will only define a single field.

Note: we may want to have separate fields for separate channels (left vs right), etc.
